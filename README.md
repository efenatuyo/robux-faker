# robux-faker
Tool that manipulates robux via http level and allows to do tasks that require robux

https://discord.gg/SwCsTNRxUC

- Buy items without spending robux
- Works on marketplace items, gamepasses, and developer products
- Supports resale marketplace purchases
- Purchase history is tracked
- Set a custom robux amount that shows in-game
- Balance persists across sessions
- Purchased items appear in your inventory
- Items show as owned everywhere
- Supports bundles and all item types
- Profile items appear in collections
- Equip items you don't own
- Equipped items stay equipped after refresh
- Works with accessories, clothing, body parts, and emotes
- Avatar renders correctly with fake items
- Everything saves automatically
- Equipped items, inventory, and balance persist between sessions
- No data loss if the proxy crashes
- Hides items you already bought from reseller lists
- Shows accurate resale prices
- Handles pending transactions
- Works with all Roblox asset types
